Rails (sort of)tutorial

2021-01-28 :: rails, ruby, tutorial

There are quite a lot of good comprehensive books on developing web applications with Ruby on Rails, like Agile Web Development with Rails 6 or The Ruby on Rails Tutorial by Michael Hartl. Although being great sources of information on building Rails applications from the ground up, they focus on a limited number of topics, omitting approaches to application architecture or skipping minutiae like disabling unwanted generators. On the other hand, there are lots of tutorials and articles on various particulars of Rails, ranging from tuning config to intricacies of the naming of ‘service objects’, but often they are short or lack practical examples.

So I want to program a tiny RoR application and with help of Github illustrate and explain steps I have made, to further use as a reference point by anyone interested or as a reminder.
